How to add data sets to Master Item (Dimension) ?

First, Lets open the Application:

  -- Right Click on "Diagnosis and Cost Dashboard"

--   Click on "Open and edit " as shown below.


Next, Locate the Code File on this page and click. The  “Master Items -Dimension” file will be downloaded.

The file will be used to populate the Dimensions in the Master Item.

Follow steps below:


From the Master-Item (Dimension) File copy and paste the Variable and Name  in the appropriate fields.


type the expression  "=State" as shown below.

Repeat and add another expression "=City" as shown below and then hit the "Add Dimension" selection then Click "Apply".

Repeat the process to another dimension as shown below.

1st select :  Drill-down.

2 expressions: =[Provider Name]

                        = Diagnosis

3. Name:     "Top Ten Providers/Diagnosis"

4 Click Add dimension

5. Click to Finalize.
